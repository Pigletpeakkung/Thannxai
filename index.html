GPT Prompt Lab

A toolkit for AI prompt engineering, cultural adaptation, and narrative automation

Overview

This project includes ready-to-use GPT prompt templates, tone-optimized writing tools, and automation scripts for product content, blogs, and chatbots. Designed for bilingual (Thai/English) use cases with cultural alignment.

Features

  • Prompt templates for product, blog, and chatbot tone design
  • Scripts for automation (Google Colab & Python)
  • Examples and screenshots of prompt outputs
  • UX writing in Thai-English context
